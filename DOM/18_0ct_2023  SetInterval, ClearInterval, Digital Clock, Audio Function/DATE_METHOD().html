Q1]what is date in javascript and what its diffrent methods expalin them with an example?

Answer:- In JavaScript, the `Date` object represents a single moment in time. It contains a set of methods that allow you to manipulate and retrieve information about the date and its objects.

Here are some of the commonly used methods of the `Date` object:


1. `getTime()`: This method returns the number of milliseconds between the specified date and midnight January 1, 1970. Example:
let currentTime = new Date(); let milliseconds = currentTime.getTime(); console.log(milliseconds);


2. `getDate()`: This method returns the day of the month (1-31) for the specified date. Example:
let currentTime = new Date(); let dayOfMonth = currentTime.getDate(); console.log(dayOfMonth);


3. `getDay()`: This method returns the day of the week (0-6) for the specified date. Example:
let currentTime = new Date(); let dayOfWeek = currentTime.getDay(); console.log(dayOfWeek);


4. `getFullYear()`: This method returns the year (4 digits for 4-digit years) of the specified date and time. Example:
let currentTime = new Date(); let year = currentTime.getFullYear(); console.log(year);


5. `getHours()`: This method returns the hour (0-23) in the specified date and time. Example:
let currentTime = new Date(); let hours = currentTime.getHours(); console.log(hours);


6. `getMinutes()`: This method returns the minutes (0-59) in the specified date and time. Example:
let currentTime = new Date(); let minutes = currentTime.getMinutes(); console.log(minutes);


7. `getSeconds()`: This method returns the seconds (0-59) in the specified date and time. Example:
let currentTime = new Date(); let seconds = currentTime.getSeconds(); console.log(seconds);

8. `getMilliseconds()`: this megod return the milliseconds in the specified date and time. Example:
let currentTime = new Date(); let Milliseconds = currentTime.getMilliseconds(); console.log(Milliseconds);
